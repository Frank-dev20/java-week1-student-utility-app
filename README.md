# Student Utility App

This is a basic student utility app that allows users to perform the following actions: 
1. Add two numbers
2. Check if a number is even or odd
3. Print a number from 1 to N (N- represent the chosen number)
4. Display day of the week from a given number
5. And exit the program by entering a particular number

## To Run the program
1. Clone this [repository](https://github.com/Frank-dev20/java-week1-student-utility-app.git)
2. Open the folder up in your IDE
3. Run the programme by click on the play button next to the `StudentUtilityApp` class.
